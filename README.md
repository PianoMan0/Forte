# Forte     
I am creating a voice assistant! 
<br><br>
Features so far:
- Jokes ("tell me a joke")
- Facts ("tell me a fact")
- Weather
- News
- Search Wikipedia ("search wikipedia for [prompt]")
- Timer
- Notes ("take note [your note]")
- Set reminders
- Convert currency
- Traffic
- Open apps (if on Windows)
- Control the volume on your computer (if on Windows)
- And more!
